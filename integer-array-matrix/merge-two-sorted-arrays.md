# Merge Two Sorted Arrays

> Description
>
> Merge two given sorted ascending integer array _A_ and _B_ into a new sorted integer array.
>
> ***
>
> Example
>
> **Example 1:**
>
> Input:
>
> ```
> A = [1]
> B = [1]
> ```
>
> Output:
>
> ```
> [1,1]
> ```
>
> Explanation:
>
> return array merged.
>
> **Example 2:**
>
> Input:
>
> ```
> A = [1,2,3,4]
> B = [2,4,5,6]
> ```
>
> Output:
>
> ```
> [1,2,2,3,4,4,5,6]
> ```
